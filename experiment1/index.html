Experiment 01

Description

Use random method to generate the different combo of text, and use jquery to setting the text in html

Technical

Core functionality is handled through the template system: JavaScript template strings with placeholder tokens (e.g. $challenge, $tool) that are replaced dynamically. Populate database: An object (populate) containing an array of context-specific terms categorized by game development roles (e.g. role), problem (problem), and tool (tool). Regular expression processing: The script uses a regular expression (/\$(\w+)/) to identify and replace all tokens in the template by: Matching any word prefixed with $ The generate() function recursively replaces tokens until there are no tokens left, then displays the result in an HTML box div. The user can regenerate the content via a button click event listener.

Reflection

I started by adapting the technical framework of the original fantasy quest generator, but completely reimagined the content, rewriting it to focus on the crises of game development. The biggest challenge was balancing technical terms (e.g. "pathfinding AI") with humorous elements ("Twitter fame"). Some filler word combinations could result in awkward phrasing, which required me to refine the word categories through iterative testing to ensure logical pairings.
