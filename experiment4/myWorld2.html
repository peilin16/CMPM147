Experiment X - Subject/Topic

Description

I adapted an example of a forest path scene that consists of the following elements: Distant background: a canopy of overlapping elliptical leaves. Near background: a grass texture created with green ovals. Path: a dynamic curved path drawn using beginShape() and vertex(), wider at the bottom and narrower at the top. Trees: a tree trunk structure generated using line segments and random angles.

Technical

I used the p5 framework to draw the leaves and grass as the base layer. The random tool of the p5 framework allows me to randomly generate a large number of points in two places to simulate leaves and grass, making the graphics fancy. I use sin function to make anime, there are two anime in this picture, first is the grass flower will random swing, second is the tree will swing when your mouse overlap the tree bottom.

Reflection

This project was a long process. I wasn't sure what to do at first, but I gradually learned about p5's drawing module and learned it through reference examples. I know that reading documentation is important, and this experience has made me realize this even more deeply. The reference materials of p5.js are invaluable resources and have been a great help to me every time I encounter difficulties.
