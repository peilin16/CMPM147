Description

In World 1, the scene presents an infinite grid, with each tile randomly displaying a Lumia character image. The distribution of Lumia is determined by the world key, meaning that different keys will generate different layouts while maintaining consistent randomness. The player can click on any tile to temporarily raise the Lumia image upwards.

Technical

World 1 uses a key-based random generation system, where the Lumia image for each tile is chosen based on a hash of the world key and the tile coordinates. This ensures consistency between key changes, while introducing randomness between different keys. The world appears infinite by dynamically recalculating the tiles as the player moves the camera.

Reflection

For World 1, World 1 meets the requirement of infinity because the blocks extend infinitely in all directions as the camera moves. The changes in space are represented by the Rumia blocks constantly changing in the world. Key changes occur when new keys are entered, which rearranges all Rumia blocks without changing the core rules. The rise animation when clicking the blocks gives signs of life, making the world feel like it is actively responding to the player's actions.
